CSS Transition + 数字计数动画 示例

示例1：CSS Transition（环形图弧长变化）

点击按钮，观察环形图弧长如何平滑过渡（不是瞬间跳变）

0 亿
原理：
- stroke-dasharray 控制环形图的弧长
- transition: stroke-dasharray 0.8s ease 让弧长变化时有0.8秒的平滑过渡
- 点击按钮后，弧长从旧值慢慢生长到新值，而不是瞬间跳变

示例2：数字计数动画（规模数字滚动）

点击按钮，观察数字如何快速滚动到新值（像计时器一样）

1000.00
原理：
- 用 setInterval 每16ms（60帧/秒）更新一次数字
- 数字从旧值逐步累加到新值，持续0.8秒
- 用户看到的是数字"滚动"效果，而不是瞬间跳变
- 类似：1000 → 1005 → 1010 → ... → 1200

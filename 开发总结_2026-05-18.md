### 2. **自算风险指标公式**
```python
# 1年收益率
year_1_return = (prices[-1] - prices[-252]) / prices[-252] * 100

# 最大回撤
peak = prices[0]
max_dd = 0.0
for p in prices:
    if p > peak:
        peak = p
    dd = (p - peak) / peak * 100
    if dd < max_dd:
        max_dd = dd

# 年化波动率
daily_returns = [(prices[i] - prices[i-1]) / prices[i-1] for i in range(1, min(n, 253))]
vol = sqrt(sum((r - avg_ret)**2 for r in daily_returns) / len(daily_returns))
annual_vol = vol * sqrt(252) * 100

# 夏普比率
sharpe = (annual_ret - 2) / annual_vol  # 无风险利率按2%计算
```

## ⚠️ 经验教训（避坑指南）

### 1. **PythonAnywhere 部署陷阱**
- **问题**: `deploy.sh` 中目录路径大小写错误
  - ❌ 错误: `cd ~/etf-tool-mvp`（大写，GitHub仓库名）
  - ✅ 正确: `cd ~/etf-tool-mvp`（小写，PA实际上目录名）
- **教训**: Linux区分大小写，部署脚本必须用 `ls ~` 先确认实际目录名

### 2. **AKShare 数据字段位置不稳定**
- **问题**: `fund_etf_spot_em()` 返回的DataFrame列位置可能变化
- **解决**: 使用列名访问（如 `df['收盘']`）而非位置索引（如 `df.iloc[2]`）

### 3. **模拟数据的危害**
- **问题**: 默认使用随机漫步生成模拟数据，所有ETF显示相同的波动率（~32%年化）
- **解决**: 优先使用本地缓存数据，模拟数据仅作为最后兜底

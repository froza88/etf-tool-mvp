# 微信公众号（订阅号）注册指引

> 个人免费，5-10分钟完成

## 准备工作

- [ ] 微信号（已实名认证）
- [ ] 邮箱（未注册过公众号）
- [ ] 身份证（本人）
- [ ] 手机号（接收验证码）

## 注册步骤

### 第一步：进入注册页面

**方式一：电脑浏览器**
1. 打开 https://mp.weixin.qq.com
2. 点击「立即注册」

**方式二：手机微信**
1. 微信搜索「微信公众平台」
2. 点击「注册」

### 第二步：选择账号类型

→ 选 **「订阅号」**

```
公众号类型：
- 订阅号（个人用，免费，每天可发1篇）← 选这个
- 服务号（企业用，每月4篇）
- 小程序 / 视频号
```

### 第三步：填写信息

1. **邮箱** → 填你的邮箱（收验证码）
2. **密码** → 设置登录密码
3. **验证码** → 去邮箱查收6位数字验证码

### 第四步：选择主体类型

→ 选 **「个人」**

### 第五步：实名认证

1. 用微信扫码
2. 填写身份证信息（姓名 + 身份证号）
3. 绑定手机号
4. 微信支付验证（扣0.01元，会退回）

### 第六步：设置公众号信息

- **名称**：你的公众号名字（建议用「ETF 对比」相关，比如：**ETF数据控** / **ETF对比实验室** / **ETF选品指南**）
- **简介**：一两句话说明账号定位（如：「不推荐，只给数据。每周用数据说话，帮你做ETF决策。」）
- **头像**：可以用简单图标（ETF相关）

### 第七步：完成

- 系统审核通过后（一般几分钟），即可登录后台发文章
- 第一次登录建议绑定**运营者微信号**（方便手机管理）

---

## 第一篇推送（发文流程）

1. 登录 https://mp.weixin.qq.com
2. 左侧菜单 → 「素材管理」 → 「新建图文」
3. 把文章内容复制进去
4. 配封面图（建议用Canva做简单封面，900x500px）
5. 预览 → 发到自己手机检查
6. 确认无误 → 群发

## 5. 结论与建议

### 5.1 总体评价

|------|------|------|
| 基础字段（代码、名称、规模、价格） | ✅ 优秀 | 100% 完整 |

| 风险字段（回撤、夏普、波动率） | ✅ 良好 | 95%+ 覆盖率 |
| 附属信息（指数名、托管人） | 🔴 待补充 | <12% 覆盖率 |

### 5.2 year_3_return 缺失处理建议

| 场景 | 建议 |
|------|------|
| **展示层** | 对 year_3_return = 0 的 ETF 标注"上市不足 3 年"或"N/A"，而非显示 0% |
| **筛选层** | 提供"仅显示有 3 年数据的 ETF"开关，避免误导 |
| **排名/排序** | 将缺失值排到末尾或排除，不参与比较 |
| **图表** | 区分"真实 0"和"无数据 0"，后者用虚线或灰色表示 |
| **数据层** | 将 year_3_return 的 0 值改为 None，更准确地表达"无数据"语义 |
